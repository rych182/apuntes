    Formas de seleccionar comparando JS vs JQuery

    JS

  • document.getElementById('id')
  • document.getElementByClassName('class')
  • document.getElementByTagName('p')

    • JQuery

  • $('#id')
  • $('.class')
  • $('p')
  • .parent() selecciona el elemento padre del id que le digas, ejemplo: $('#tercera').parent()
  • .children() selecciona los hijos del elemento seleccionado $('#padre').children() y si quieres seleccionar un hijo en particular, $('#padre').children('#tercera')
  • .siblings() selecciona los elementos hermanos. Ejemplo: $('#tercera').siblings().fadeOut();
  • .prev() Selecciona el elemento anterior. Ejemplo: $('#tercera').prev().css({});
  • .next() Selecciona el elemento siguiente. $('#tercera').next().css({});
  • .prevAll() Selecciona todos los elementos anteriores al seleccionado. Ejemplo: $('#tercera').prevAll().css({});
  • .prev() Selecciona todos los elementos siguienes al seleccionado. Ejemplo: $('#tercera').nextAll().css({});
  • .first() Selecciona el primer elemento de todos los seleccionados Ejemplo: $('.caja').first().css({});
  • .last() Selecciona el ultimo elemento de todos los seleccionados Ejemplo: $('.caja').last().css({});
  • .eq() Selecciona el elemento por su indice, el 0 es el 1 y el 1 es el 2 Ejemplo: $('.caja').eq(0).css({background: '#000'});
  • .filter() Selecciona el elemento que le indiques y puedes seleccionar más cajas a diferencia de eq(). Ejemplo: $('.caja').filter(".naranja").css({background: '#000'});
  • .not() Selecciona todas las cajas menos la que le indiques. Ejemplo: $('.caja').not(".naranja").css({background: '#000'});

    • encadenar en jquery

  • $('#tercera').parent().css({})

    Funciones de Jquery

  • .fadeOut(500); desaparece los elementos en medio segundo. ejemplo: $('#padre').children().fadeOut(500);
  • .slideUp(500); desaparece los elementos hacia arriba en medio segundo $('#padre').children().slideUp(500);
  • .find() te encuentra etiquetas, clases y ids. Ejemplo: $('#padre').find('div.caja').slideUp(500);
