    Pasos para usar

  1. Adjuntar libreria
  2. Añadir la directiva ng-app en tu etiqueta html y ponerle un nombre
  3. Inicializas la aplicacion: var cualquierNombre = angular.module('expresiones', []);
  4. Creas tu controlador:

    cualquierNombre.controller('miControlador', function($scope) {

    $scope.mensaje = "Hola mundo ";

    });

    OJO: no dejes espacios al declarar el controlador en el HTML o no funcionara
