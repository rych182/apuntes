Section con divs dentro

El Display: inline; abarca el contenido que contenga la etiqueta y se pone uno al lado del otro, no importa que tu le hayas puesto un width y height especificas también si el contenedor padre tiene display: inline, los dos contenedores mayores se pondran uno al lado del otro

  • Las propiedades width y heigt no funcionan cuando usas display:inline; con divs, P , etiquetas H, span
  • Si quieres modificar el ancho de una caja que tiene display: inline, puedes usar la propiedad padding
  • Las propiedades width y height. SI tienen efecto en las etiquetas IMG
