display: inline-block

Inline-block: este se comporta como un elemento en línea (inline) pero también puede tener dimensiones (ancho y alto) y aceptar márgenes y rellenos (margins and paddings), al igual que un elemento de bloque (block).

Se utiliza comúnmente para crear diseños de página en los que se desea que los elementos fluyan uno al lado del otro

Ejemplos de su uso:

  • Creación de galerías de imágenes: Al utilizar "display: inline-block" en los elementos de imagen, se pueden mostrar varias imágenes en una fila, con la capacidad de ajustar el tamaño y los márgenes de cada imagen.
  • Creación de menús de navegación horizontales: Al aplicar "display: inline-block" a los elementos del menú, se puede lograr que se muestren uno al lado del otro en una sola línea.
  • Creación de elementos de formulario personalizados: Al aplicar "display: inline-block" a los elementos de formulario, como cajas de verificación (checkboxes) o botones de opción (radio buttons), se pueden alinear en una sola línea.
