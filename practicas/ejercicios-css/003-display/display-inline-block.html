2 Sections con 3 divs cada uno

  • El section es el elemento padre que tiene un display:inline-block para que así cada section se ponga uno al lado del otro
  • El section debe de tener una medida de ancho la cual haga que quepan todos sus elementos, de lo contrario recorreran hacia abajo al elemento hijo(algún div)
  • El tamaño de los hijos(div) no se ve afectado cuando el elemento padre (section) modifica su tamaño
  • Cuando el ancho o el alto no tiene medidas específicas, crecera todo lo que necesiten crecer por default
