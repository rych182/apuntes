Todos los elementos tienen un "position static" por default, todo lo que tenga un position relative, absolute o fixed, son elementos "posicionados"


Cuando utilizas "position: relative", desbloqueas 4 metodos de CSS que son de desplazamiento:4 TOP, BOTTOM, LEFT y RIGHT, esto permite desplazar al elemento "sin desplazar" a los demás elementos como en el caso de "la propiedad margin"


Al usar "position relative" y moverlo con TOP, BOTTOM, LEFT y RIGHT, este se "encima" sobre elementos que tienen "position static"


Si todos los elementos tuvieran "position relative" el elemento de arriba se pone DEBAJO del siguiente elemento con position relative


Si TODOS tienen "position: relative", el primero, "se pondría debajo" de los "siguientes", esto se debe a que todos los elementos tienen una mayor relevancia en el Eje Z


Si quisiera poner por encima un elemento que tiene position: relative encima del siguiente elemento que también tiene position: relative debo de usar la propiedad z-index con un número positivo, el z-index se desbloquea solo cuando los elementos tienen un position DISTINTO al position: STATIC
