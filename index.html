  • ¿Qué es el modelo de cajas? Es un módulo CSS que define cajas rectangulares, incluyendo sus rellenos y márgenes, que son generadas para los elementos y que se disponen de acuerdo al modelo de formato visual. Nos permite establecer el diseño de nuestro sitio web
  • ¿Qué es BEM? es una metodología que le ayuda a crear componentes reutilizables y compartir código en el desarrollo front-end. Hace que su código de front-end sea más fácil de leer y comprender, más fácil de trabajar, más fácil de escalar, sólido, explícito, y mucho más estricto. Garantiza que todos los que participan en el desarrollo de un sitio web trabajen con una única base de código y hablen el mismo idioma. El uso de la convención de nomenclatura adecuada de BEM lo preparará mejor para los cambios de diseño realizados en su sitio web.

  • ¿Qué es Flexbox? Es un método que pueda ayudar a distribuir el espacio entre los ítems de una interfaz y mejorar las capacidades de alineación.
  • ¿Para que sirve flex-wrap? es una propiedad del contenedor flex y especifica si puede haber un cambio de línea ( wrap pone las cajas que no quepan, abajo ) o no ( nowrap ). El valor por defecto es nowrap.
  • ¿Para que sirve justify-contend? distribuye el espacio (flex-start, flex-end, space-between, space-around).
  • Diferencia entre space-between y space-around: space-around deja un espacio en las orillas
  • Diferencia entre justify-content y justify-items:

      justify-content: along primary axis (horizontal alignment/spacing if flex-direction is row or vertical alignment/spacing if flex-direction is column)

    1. flex-start Alinear a los hijos horizontalmente a la izquierda
    2. flex-end Alinear a los hijos horizontalmente a la derecha
    3. center Alinear a los hijos centrados horizontalmente (¡asombro!)
    4. space-between Distribuya a los hijos horizontalmente de manera uniforme en todo el ancho.
    5. space-around Distribuya a los hijos horizontalmente de manera uniforme en todo el ancho (pero con espacio en los bordes

      align-items: a lo largo del eje secundario (establezca la alineación vertical si la dirección de flexión es una fila u alineación horizontal si la dirección de flexión es la columna)

    1. flex-start Alinear a los hijos verticalmente arriba
    2. flex-end Alinear a los hijos verticalmente hacia abajo
    3. center Alinear a los hijos centrados verticalmente (¡asombro!)
    4. baseline Los hijos alineados verticalmente para que sus líneas de base se alineen (realmente no funciona)
    5. stretch Obligar a los hijos a estar a la altura del contenedor (ideal para columnas)
